Window

- It is the root level element in any web page.
- All the global variables are defined on the window object.
- For example, alert(), confirm() are methods defined on the window object.

Document

- It is the direct child of the window object.
- It is known as Document Object Model (DOM).
- You can access it via window.document or document.
- Document Object has many useful methods defined on it. For example, document.getElementById(), document.getElementByTagName(), document.createElement(), document.querySelector() and many more.
